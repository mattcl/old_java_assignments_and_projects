Editor interface overview

top

Opening an existing document

Select "File" -> "Open" from the menu. Select the desired directory containing your document.

top

Saving a document

Select "File" -> "Save" from the menu. If you have not already saved this document, you will be prompted to specify a name for the directory.
You may also select "File" -> "Save As" to save to a different location.

top

Adding/removing pages

To add a page, click on the "Add Page" button in the eidtor toolbar.
To remove a page, select a page from the catalog and click the "Remove Page" button in the editor toolbar.

top

Adding/removing shapes

To add a shape, click on the "Add Shape" button in the editor toolbar.
To remove a shape, select a shape (either from the canvas or from the catalog) and click the "Remove Shape" button in the editor toolbar.

top

Adding/removing resources

To add a sound file (.au ONLY), click on the "Add Sound" button in the editor toolbar and select the desired file. The sound resource does not need to be in your current document's directory.
To add an image file (.jpeg or .gif), click on the "Add Image" button in the eiditor toolbar and select the desired file. Like with the sound resources, the image file does not need to be in your current document's directory.

To remove a resource, select the resource from the catalog and then click the "Remove Resource" button that appears in the inspector panel.

top

Manipulating objects

To change the properties of a page or shape, or to inspect an image or sound clip, select the desired object (either from the catalog or from the canvas) and manipulate the properties in the inspector panel.
When editing pages and shapes, pressing "enter" after altering a property will cause the change to take effect.

top

Scripting

Shapes are scriptable, and scripts may be entered in the script field of the shape inspector panel. Press "enter" for the script to take effect.

top

Playing the game

To play the game, simply press the "Play" button in the upper right on the toolbar.
